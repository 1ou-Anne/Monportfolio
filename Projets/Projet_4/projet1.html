Définition

La régression linéaire :

La régression linéaire est une technique statistique utilisée pour modéliser la relation entre une variable dépendante continue et une ou plusieurs variables indépendantes. Elle est souvent utilisée pour prédire ou estimer la valeur d'une variable basée sur les valeurs des autres variables. La régression linéaire suppose que la relation entre les variables est linéaire, c'est-à-dire que le changement dans la variable dépendante est proportionnel aux changements dans les variables indépendantes. La forme générale d'un modèle de régression linéaire simple peut être exprimée par l'équation suivante :
Y = β0 + β1 × X + ε


Le Fat Free Mass Index :

Le Fat Free Mass Index (FFMI) est un indice utilisé pour évaluer la masse maigre dans le corps humain. Le FFMI se concentre spécifiquement sur la masse maigre, c'est-à-dire la masse musculaire sans tenir compte de la graisse corporelle.
